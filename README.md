### Usage

- `git clone https://github.com/samsoft00/acumen-task-nodejs.git acumen-oye-task`
- `cd acumen-oye-task`
- `npm i`
- `cp .env.example .env`
- `npm run start:dev`


**To start server Development**

```
npm run start:dev
```

**Run test**

```
npm run test
```
